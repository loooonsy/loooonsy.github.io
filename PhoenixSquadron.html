Phoenix Squadron

Phoenix Squadron was my first "real" GIMM project. It is a simple Adobe Animate game, where enemies scroll down the screen while you try to destroy them before they can make it past you. The premise is that when Voyager was launched, it contained only a VHS of the movie Top Gun. It would be found by an alien civilization, which then built their military to mimic the film. Almost immediately after Sunrise, we were given the task of building a fully functioning game using Adobe Animate. Phoenix Squadron was my response to that challenge, and represents a milestone in my abilities to adapt and work with still unfamiliar software, and practicing good code structure.
